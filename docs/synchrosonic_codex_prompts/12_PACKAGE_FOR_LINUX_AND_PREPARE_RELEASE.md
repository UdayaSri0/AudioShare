# Prompt 12 — package for Linux and prepare open-source release

```text
Prepare the application for a real Linux open-source release.

Goal:
Make the project buildable and releasable for Linux users.

Requirements:
- clean release profile
- app metadata
- desktop entry
- icon handling
- packaging plan for:
  - native Linux build
  - AppImage
  - .deb
- CI workflow for build/test/lint/package
- release checklist
- contribution guide
- issue templates
- basic security/reporting policy
- About dialog metadata

Important:
- do not invent fake release artifacts
- implement what can be done in the repo now
- document any remaining packaging gaps honestly

Deliverables:
- GitHub Actions or equivalent CI
- packaging docs
- release checklist
- CONTRIBUTING.md
- LICENSE presence check
- polished README

Validation:
- run build/test steps
- summarize release readiness and remaining blockers
```
